Дана функция:

\begin{equation*}
	y(x) = 
	\begin{cases}
		2 \cdot x - 10, & x > 0 \\
		0, & x = 0 \\
		2 \cdot \left| x \right| - 1, & x < 0
	\end{cases}	
\end{equation*}

Необходимо вычислить ее значение по заданному агрументу.

\InputFile

$X$~---~целое число, которое является аргументом функции

\OutputFile

$Y$~---~значение функции
